# Book-Store
 
## Project Overview :-
This is a Book Store App where we can see all the books and add books accordingly.

## Tech-stack Used :-
* Javascript
* CSS
* NodeJS
* EJS
* MongoDB
* React

## Libraries Used :- 
* ExpressJS
* Mongoose
* Material UI

## Created Pages :- 
* Home Page
* NavBar
* Add Book Page
* List of Books Page
* Payment

## Functionalities Added :- 
* The data is fetched from a cloud database of MongoDB - Atlas
* Search Filter by name
